# حاسبة ومخطط المولد التزامني (Alternator Phasor Calculator)

تطبيق ويب تفاعلي لحساب ورسم مخطط المتجهات للمولد التزامني (Alternator/Synchronous Generator).

## المميزات
- حساب التيار والجهود في المولد التزامني
- رسم مخطط المتجهات بشكل تفاعلي
- دعم حالات عامل القدرة المتأخر والمتقدم
- واجهة مستخدم عربية سهلة الاستخدام

## التقنيات المستخدمة
- React
- TailwindCSS
- HTML5 Canvas

## التشغيل محلياً
1. قم بنسخ المستودع
2. افتح ملف `index.html` في المتصفح

## النشر على GitHub Pages
1. ارفع الملفات إلى مستودع GitHub
2. اذهب إلى Settings > Pages
3. اختر الفرع الرئيسي (main) كـ source
4. احفظ الإعدادات

## ربط الدومين
1. أضف ملف `CNAME` في المستودع مع اسم الدومين الخاص بك
2. قم بتحديث إعدادات DNS للدومين ليشير إلى GitHub Pages
